What FakeImage Detector does

A lightweight web experience powered by a segmentation model trained on AI-inpainted images.

Upload any image

Drop in a JPG or PNG and the backend model runs inference in seconds.

Highlight edited regions

Pixel-wise masks overlay suspicious areas – insertions, removals, and heavy edits.

Downloadable evidence

Export side-by-side views for claim files, reports, or case notes.

How it works

Under the hood, FakeImage Detector is a segmentation model tuned for AI inpainting artifacts.

1. Train on AI-edited datasets

Beyond-the-Brush + COCO-Inpaint image/mask pairs with realistic edits.

2. Learn a pixel-wise mask

U-Net & Vision Transformer models are trained to segment altered regions.

3. Serve via web UI

User uploads an image → model predicts mask → UI overlays results in-browser.
